Online Security

By Michael Teixeira

There are a number of things you can do to increase your security while using the internet, varying from basic security measures which should be practiced by everyone ideally, and extreme security measures which as you can guess are extremities that most people can do without. We will stick with the former measures for now so that you may learn and or touch up on the very basic standards of security whilst using the internet.
